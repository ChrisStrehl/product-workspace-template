# templates/ — Standard Formats

Reusable formats for the artifacts you produce, so every PRD/story/epic looks the same — and so skills know exactly what to generate.

Suggested: `prd.md`, `User Stories.md`, `Epic.md`, `bug tickets.md`, `bet.md`.

Add the templates your team actually uses. Skills reference these when they create documents.
